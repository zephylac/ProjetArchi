# ProjetArchi

# Install

## Ubuntu
	```
	curl -sL https://deb.nodesource.com/setup_9.x | sudo -E bash -
	sudo apt-get install -y nodejs
	./install.sh
	```

## ArchLinux
		```
		sudo pacman -S nodejs npm
		./install.sh
		```
