# Install

## Ubuntu
curl -sL https://deb.nodesource.com/setup_9.x | sudo -E bash -

sudo npm install -g npm@latest

sudo npm install -g @angular/cli

## ArchLinux
`
sudo pacman -S nodejs npm; ./install.sh
`
